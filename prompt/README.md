# Prompt Files Index

This folder contains planning and prompt documents for the FootDash project.

- `001-concept-description.md` — Concept description (v1)
- `002-concept-overview.md` — High-level concept overview
- `003-concept-product-definition-document.md` — Product definition document (PDD)
- `004-phase-1-development-plan.md` — Phase 1 development plan (MVP)
- `005-phase-1-technical-architecture-diagram-stack explanation.md` — Technical architecture overview and stack explanation
- `006-architecture-as-a-visual diagram.md` — Visual architecture diagram (PNG embedded)
- `007-phase-1-scaffold-plan.md` — Phase 1 scaffold plan with CLI commands and folder structure

How to use

- These files provide a starting point for scaffolding the frontend and backend projects.
- Follow the CLI commands in `007-phase-1-scaffold-plan.md` to create starter projects.
- Add more prompt files here as the project evolves.
