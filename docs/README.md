# FootDash Documentation

This directory collects living documents that guide the evolution of the FootDash platform.

## Contents

- **`FEATURE_ROADMAP.md`** – comprehensive feature roadmap with detailed implementation plans, acceptance criteria, and sprint calendar (Phases 8–12).
- `migration-roadmap.md` – step-by-step plan to migrate the current Express + Ionic stack toward the Phase 1 scaffold (NestJS + modular Angular).
- `architecture/` – copy of the Phase 1 scaffold blueprint, development plan, architecture overview, and supporting diagrams.
- `features/` – individual feature implementation plans from completed phases.
- `ci-cd-pipeline.md` – CI/CD pipeline documentation and configuration guide.
- `phase-7-ui-ux-enhancements.md` – UI/UX polish and navigation fixes (Phase 7).
- `phase-e-checklist.md` – Phase E cleanup checklist.
- `phase-e-lessons-learned.md` – Retrospective and lessons from Phase E.
- `ops/` – operational runbooks and deployment guides.
- `setup/` – development environment setup instructions.
